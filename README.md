# PDFGenZ PlatformOps Lab

A complete DevOps and Platform Engineering portfolio project demonstrating Docker, Kubernetes, Helm, ArgoCD, GitHub Actions, monitoring, logging and DevSecOps concepts.

## Project Goal

This project shows how a modern application moves from code to deployment using DevOps automation.

## Architecture

Developer Laptop  
→ GitHub  
→ GitHub Actions  
→ Docker Build  
→ Kubernetes  
→ Helm  
→ ArgoCD GitOps  
→ Monitoring with Prometheus/Grafana  
→ Logging with ELK concepts  
→ Security with DevSecOps practices

## What Each Tool Does

### GitHub
Stores application source code and version history.

### GitHub Actions
Runs CI/CD pipelines automatically when code is pushed.

### Docker
Packages the application into a portable container image.

### Kubernetes
Runs and manages containers reliably.

### Helm
Packages Kubernetes YAML files into reusable charts.

### ArgoCD
Implements GitOps by automatically syncing Kubernetes with Git.

### Prometheus
Collects application and infrastructure metrics.

### Grafana
Displays dashboards and alerts.

### ELK Stack
Collects and searches application logs.

### DevSecOps
Adds security checks into the development and deployment process.
